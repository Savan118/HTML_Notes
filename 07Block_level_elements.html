Block Level elements

        < address >
        This Tag is used to show Contact information.
        
        < article >
        Used to show Article content.
        
        < aside >
        Used to display Aside content.
        
        < blockquote >
        Long ("block") quotation.
        
        < details >
        Disclosure widget.
        
        < dialog >
        Dialog box.
        
        < dd >
        Describes a term in a description list.
        
        < div >
        Display a Document division.
        
        < dl >
        Description list.
        
        < dt >
        Description list term.
        
        < fieldset >
        Field set label.
        
        < figcaption >
        Figure caption.
        
        < figure >
        Groups media content with a caption (see < figcaption >).
        
        < footer >
        Section or page footer.
        
        < form >
        Input form.
        
        < h1 >, < h2 >, < h3 >, < h4 >, < h5 >, < h6 >
        Heading levels 1-6.
        
        < header >
        Section or page header.
        
        < hgroup >
        Groups header information.
        
        < hr >
        Horizontal rule (dividing line).
        
        < li >
        List item.
        
        < main >
        Contains the central content unique to this document.
        
        < nav >
        Contains navigation links.
        
        < ol >
        Ordered list.
        
        < p >
        Used to display a text as paragraph.
        
        < pre >
        Used to show Preformatted text.
        
        < section >
        Section of a web page.
        
        < table >
        This is a Table tag to diplay a table.
        
        < ul >
        Unordered list.
