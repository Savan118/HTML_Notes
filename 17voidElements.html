HTML Void elements are elements that do not have a closing tag, meaning they only have an opening tag. These elements are used to insert non-textual content into an HTML document, such as images, line breaks, and input fields.

Here is a list of some common HTML Void elements:


<img>: Used to insert an image into a web page. The source of the image is specified in the src attribute.

Example: < img src="image.jpg" alt="An image">


<br>: Used to insert a line break into a web page.

<input>: Used to create an input field where users can enter data.

Example: < input type="text" name="username">


<meta>: Used to provide metadata about an HTML document, such as the character encoding and keywords for search engines.

Example: < meta charset="UTF-8">


<link>: Used to link an HTML document to an external resource, such as a CSS file.

Example: < link rel="stylesheet" href="style.css">
